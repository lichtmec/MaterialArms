# MaterialArms
Minecraft Mod  
カスタマイズが可能なツールや防具、またそれを作成・運用するための工場設備を追加します。  
<br />
本リポジトリでは、ビルド済みのバイナリを配布します。  
APIのソース、DevバイナリはAPIリポジトリから取得してください。  
https://github.com/lichtmec/MaterialArms-API  
<br /><br />

## System requirements  
- Minecraft 1.7.10  
- Forge 1448+  
<br />

## Supported  
- NotEnoughItems
- IndustrialCraft2　2.2.746+
- GregTech　6.05.22+
